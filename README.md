# react-parcel-boilerplate

## Features:
* Parcel 1.11 
* Babel 7
* React 16.8
* React Router v4
* Prop-types
* Sass
* ESLint (with React config and Prettier plugin)
* Husky + Lint-staged

## Scripts
* **dev**: Run dev build with server instance,
* **build**: Run production build,
* **watch**: Run file watcher without server instance
